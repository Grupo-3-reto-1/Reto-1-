INSTALACIÓN DEL PROXMOX

Lo primero que neciesitaremos para poder crear nuestro servidor es instalar Proxmox Para ello deberemos Descargar la iso de la sigiente pagina. https://www.proxmox.com/en/downloads Una vez tengamos la iso debemos crear un USB booteable para ello utilizaremos Etcher El siguinte paso tendremos que insertar el usb booteable en la maquina en la que queremos instalar el proxmox y cambiar el orden de arranque.
El proximos pasos se basan en la configuracion del proxmox. En esta guia enseñaremos nuestra configuracion. Lo primero que tenemos que hacer es decidir la particion donde queremos que se instale el proxmox. Ahora toca decidir las caracteristicas que queramos que tenga como la IP del servidor y su gateway como la contraseña y el puerto de acceso entre otros.

Una vez terminada la instalacion acabaremos en esta pantalla que nos dice que reiniciemos y entremos desde otra maquina antes de reiniciar. Es recomendable hacer una foto a esta pantalla ya que es muy importante acordarnos de las configuraciones que hemos hecho.

INSTALACIÓN DEL SERVIDOR WEB

Habiendo completado la instalacion reinicimos la maquina
Despues de reinciar debemos entrar a la ip de la maquina Proxmox utilizando otra maquina en la misma red llamaremos a esta maquina "cliente"
Desde el cliente empezaremos el proceso de creacion de una nueva maquina virtual

Para ello tendremos que subir una iso del sistema que queramos utilizar para el servidor web (en nuestro caso utilizaremos Ubuntu 16.4)
Una vez hayamos subido la iso debemos especificar las caracteristicas de la maquina teniendo en cuenta las limitaciones fisicas de la maquina Proxmox
Lo siguiente sera instalar el sistema operativo de manera habitual

INSTALACIÓN DEL APACHE2


Habiendo instalado el sistema operativo debemos instalar y configurar el servicio apache
Para ello abriremos la consola de comandos de linux y escribimos los siguentes comandos:


apt-get update
apt-get upgrade
sudo apt-get install apache2 apache2-doc apache2-utils



Una vez instalado debemos abrir el archivo de configuracion mediante el comando

sudo nano /etc/apache2/apache2.conf


Dentro del archivo de confiuracion añadimos esta linea al final del todo ServerName Dominio_o_ip
Hacemos un sudo apache2ctl configtest para asegurarnos de que todo funciona correctamente
Y al final reincimaos el apache con el comando:

sudo systemctl restart apache2


Tambien debemos asegurarnos de que el firewall esta bien configurado con:

sudo ufw app info "Apache Full"


ejecutamos el comando sudo ufw allow in "Apache Full" para permitir el trafico entrante
ya podemos entrar a la pagina web utilizando http://ip del servidor
